# NLP Chatbot for Restaurant 🍽️

This project is an AI-based chatbot developed using Google Dialogflow and Flask for restaurant order management.

## 🚀 Features
✅ NLP-based chatbot (Google Dialogflow)  
✅ Order Management System (Place, Track Orders)  
✅ Flask Backend with MySQL Database  
✅ Static Files Handling (CSS, JS, Images)
